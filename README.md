## Note
- If you want hyprland config, you only need ags dan hypr folder.
- Something may not work for you, I used this configuration in my used Thinkpad x270 and didn't think about dual monitor.

### Dependencies
- hyprland (ofc)
- ags
- wf-recorder
- slurp
- wayshot
- powerprofilesctl
- sassc
